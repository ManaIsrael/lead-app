# Lead Management App

Simple full-stack app to manage leads.

## Tech Stack
- Frontend: Next.js
- Backend: Node.js, Express
- DB: MongoDB (Mongoose)

## Deployment
- Frontend on: Vercel
- Backend on: Render
- Vercel Link: https://lead-app-indol.vercel.app/

### Backend
```bash
cd backend
npm install
# Add .env with your MONGO_URI
npm run dev
